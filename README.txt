README: How to Install the Football Gambler Game
This guide provides step-by-step instructions for installing the Football Gambler game on Windows 10, Windows 11, and macOS. Please follow the instructions specific to your operating system.

Installation Instructions for Windows 11:
Ensure Python is installed on your computer. If not, download it at: https://www.python.org/downloads/
Locate the “FootballGambler” folder in your downloads or saved location.
Right-click the folder, select “Open in Terminal.”
Copy and paste “python FootballGambler.py” into the Terminal or Command Prompt.
Press Enter, and the game should start.
For further assistance, check the support folder.

Installation Instructions for Windows 10:
Ensure Python is installed on your computer. If not, download it at: https://www.python.org/downloads/
Locate the “FootballGambler” folder in your downloads or saved location.
Left-click the folder, hold down Shift, and select “Copy as path.”
Open the Command Prompt by searching and selecting the first option.
Type "cd " in the command prompt, right-click to paste the path, and press Enter.
Copy and paste “python FootballGambler.py” into the Command Prompt.
Press Enter, and the game should start.
For further assistance, check the support folder.

Installation Instructions for macOS:
Ensure Python is installed on your computer. If not, download it at: https://www.python.org/downloads/
Locate the “FootballGambler” folder in your downloads or saved location.
Right-click the folder and select “New Terminal at Folder.”
Copy and paste “python FootballGambler.py” into the Terminal.
Press Enter, and the Football Gambler game should start.
